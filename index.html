Preguntas frecuentes

Antes de empezar

¿Necesito tener un CV anterior?

No. Si tenés uno, sirve como referencia. Si no, el formulario guía la información desde cero.

¿Se garantiza conseguir trabajo?

No. El servicio mejora la presentación de tu perfil, pero ninguna postulación puede garantizar entrevistas o contratación.

¿Cómo se entrega?

La app debe permitir descarga digital del CV generado. Según el plan, también puede incluir versión editable.

¿Puedo revisar antes de descargar?

La versión automática debe mostrar una vista previa editable para corregir datos antes de generar el archivo final.

¿Cuántas veces se genera el CV?

Una sola vez por compra. Antes de confirmar, la app debe mostrar validaciones y una vista previa editable para evitar errores.

¿Cuánto tarda?

El objetivo del producto es generar el CV en el momento o en pocos minutos después de confirmar el pago y completar los datos.

¿Qué pasa después de pagar?

La app debe validar el pago automáticamente y habilitar la generación del CV para ese pedido.

¿Qué medios de pago se aceptan?

Los disponibles en Mercado Pago: tarjetas, dinero en cuenta y otros medios habilitados por la plataforma.

¿Puedo cancelar?

Si el trabajo personalizado todavía no comenzó, se puede solicitar cancelación por el canal de contacto. Si ya fue entregado, se aplica la corrección incluida.

¿La IA puede inventar datos?

El sistema debe usar solo la información cargada por la persona. Antes de descargar, se muestra una vista previa editable.

¿Qué incluye el plan Enfocado?

Permite cargar el puesto, la empresa y el aviso laboral para priorizar habilidades y experiencia real que coincidan con esa búsqueda.

¿Qué es una versión ATS-friendly?

Es una versión simple, clara y de una columna, pensada para que sistemas automáticos de selección puedan leer mejor el contenido.

Soporte

¿Tuviste un problema con el pago o la descarga?

Si algo falla, enviá el comprobante de pago y un contacto para revisar el pedido manualmente.

Privacidad

Uso de datos personales

Los datos y archivos enviados se utilizan exclusivamente para generar el CV solicitado y gestionar ese pedido, conforme a la Ley 25.326 de Protección de Datos Personales. No se venden ni se publican respuestas o archivos.

Datos como DNI, teléfono, email o dirección pueden cargarse de forma opcional y el usuario debe poder elegir si se muestran en el CV final. No se solicitan datos sensibles como salud, religión, afiliación política o sindical. Para solicitar acceso, corrección o eliminación de datos, escribir al correo de contacto del servicio.
